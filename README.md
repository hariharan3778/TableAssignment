# Table Generator
This is a simple Java program that generates a P × P table using random even numbers from 2 to 16. The user can enter an even number, and the program highlights all occurrences of that number in the table. It also displays how many times the selected number appears.
## Features
- Generates a P × P array.
- Uses random even numbers between 2 and 16.
- Displays the table in the terminal.
- Highlights the selected even number.
- Counts and prints the total occurrences.
- Handles invalid user input.
## Requirements
- VS Code or any Java IDE
## How to Run
Compile the program:
```bash
javac Table.java
```
Run the program:
```bash
java Table
```
